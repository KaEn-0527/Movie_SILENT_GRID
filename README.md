# SILENT GRID — 映画公式ランディングページ

近未来SFサスペンス映画『SILENT GRID』（架空作品）の公式LP。
Figmaデザインをもとに、依存ライブラリなしの HTML / CSS / JavaScript で実装しています。

> この街では、すべての記憶が監視されている。
> 彼女との最後の3日間だけが、消されていた。

## 特徴

- **バニラ実装** — フレームワーク・ビルド不要。静的ファイルのみで動作
- **FVイントロ映像** — 初回に映像を再生し、最終フレーム（キービジュアル）で静止 → 見出し・CTAが立ち上がる演出。省データ／低速回線／`prefers-reduced-motion` では自動でスキップ
- **タイプライター演出** — 見出し・ラベルを1文字ずつ表示（読み上げ・検索・コピーは通常どおり）
- **記憶ログの復元演出** — 「記憶ログを復元する」で欠落した3日分に文字化けした断片が復元される
- **場面写真ライトボックス** — オーバーレイ表示、前後移動（← →）対応
- **上映館検索** — 地域選択・現在地判定による上映館／上映時間の表示
- **公開フェーズ切替** — 劇場公開／配信フェーズで CTA を出し分け
- **計測設計** — 主要CTA・スクロール到達率・滞在時間などを `dataLayer` へ送出
- **アクセシビリティ / パフォーマンス** — キーボード操作、フォーカストラップ、`prefers-reduced-motion`、画像 WebP 化・遅延読込

## 構成

```
index.html              マークアップ
assets/css/style.css    スタイル（デザイントークン〜各セクション）
assets/js/main.js       挙動（依存なし）
public/images/          画像（WebP）
public/video/           FV映像（最適化済み MP4：デスクトップ / モバイル）
```

## ローカルでの確認

任意の静的サーバーで配信します。

```bash
npx serve .
# http://localhost:3000 を開く
```

### 確認用クエリ

| クエリ | 挙動 |
|---|---|
| `?intro=0` | FVイントロ映像をスキップ |
| `?phase=stream` | 配信フェーズを初期表示 |
